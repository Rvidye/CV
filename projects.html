Multiplayer Third Person Shooter using Unreal Engine 4. (09/2018 - 04/2019)

A simple third person shooter project made in unreal engine 4 to demonstrate computer networking fundamentals (TCP/IP) stack. Made using blueprints only. Player can host sessions on Lan environment and other players can join. Deathmatch and classic modes are available to players.

Stealth Prototype using Unreal Engine 4. (UE42020 Spring Jam 06-2020)

A simple prototype for stealth-based game where player can turn invisible for limited time. Based on metal gear series lore. All enemies are implemented using behavior tree in unreal engine. Game is made using 100% blueprints. All particles and visual effects were made using Unreal Engine 4. Due to lack of time only 3 levels were made.

VR_Speech_Anlyzser Using Unity Engine(2019-2020)

A simple android VR app made in Unity Engine for user to practice public speaking. Focused to help people with glossophobia. Android app is connected to a cloud server, where speech is uploaded and then analyzed to give user score. Also published IEEE paper for above project. Received Best IEEE Paper award from AICTE IEEE Committee.
